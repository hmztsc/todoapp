## About
Todo App with PHP Laravel 8/9.

## Installation
- composer update
- npm install
- create a new database from your host collation: utf8mb4_unicode_ci
- rename .env.example -> .env and set database informations
- php artisan migrate
- php artisan db:seed
- php artisan key:generate
- php artisan serve
